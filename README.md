# Quiz-website

A responsive, interactive quiz application built with HTML, CSS, and JavaScript.
This project features a modern, colorful UI on a sleek black background, supports PC, tablet, and mobile, and includes an automated countdown timer for each question.

Features
Responsive Design

Fully optimized for desktop, tablet, and mobile phones
Clean layout with a colorful UI against a dark theme

10-Second Timer

Each question comes with a 10-second countdown
If no option is selected before time runs out, the quiz automatically moves to the next question

Interactive Options

The Next button is disabled by default
It becomes enabled only after selecting an option, preventing accidental skipping

Score Calculation

User answers are tracked throughout the quiz
A result page displays the final score at the end of the quiz

Technologies Used

HTML5 – Structure of the quiz pages
CSS3 – Styling, animations, and responsiveness
JavaScript (ES6) – Logic for timer, navigation, scoring, and interactivity

How It Works

-The quiz loads the first question with a 10-second timer
-User selects an option → Next button activates
-Timer reaches zero → quiz auto-moves to the next question
-After the final question, a result summary is shown
